# Codex 用要件メモ

## 守ること

- Git 前提で変更を積み上げる
- `.ids` / DB / 出力ファイル / `.venv` を追跡しない
- 大きい変更は `feature/*` ブランチで進める
- 破壊的変更前にコミットを挟む

## 実装優先順位

1. `.ids` parser
2. SQLite 構造
3. バッチ実行計画
4. 自動紐付け
5. 平均ボルタモグラム

## MVP 範囲

- MIP 登録
- MIP 使用記録登録
- セッション登録
- 条件登録
- バッチ計画生成
- 測定追加
- `.ids` 取り込み
- 条件抽出
- 集計
- 平均ボルタモグラム
- CSV / Excel / PNG / Markdown 出力

## 拡張項目

- 複製
- 横断比較
- 監視 UI
- レポート出力
- failed 再キュー化
